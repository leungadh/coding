Implementing plot with

  • Color
  • Linestyle
  • Linewidth
  • Marker, Marker size
  • Label
  • Legend
x = np.linspace(0, 20)
y1 = np.sin(x)
y2 = np.sin(x - np.pi)

plt.figure()

plt.plot(x,
         y1,
         color='black',
         linestyle='-',
         linewidth=2,
         marker='s',
         markersize=6,
         label='y1')

plt.plot(x,
         y2,
         color='red',
         linestyle='--',
         linewidth=2,
         marker='^',
         markersize=6,
         label='y2')

plt.legend()
;
